二、回合流程详细时序

1

🎮 回合开始（runTurn()）

round += 1 · 检查level是否变化 · 检查是否超过最大回合数（20）

2

🏠 【灯塔阶段】（lighthousePhase()）

渲染船员卡片（5个） · 等待玩家分配 · 结束回合

3

🌩️ 【风暴阶段】（stormPhase()）

风暴压力+1 · 风暴之眼移动 · 生成风暴格子 · 爆炸结算 · 亮度-1

4

🏆 【胜负判定】（checkVictory()）

胜利条件：VP达标 或 存活10回合 · 失败条件：结构≤0 或 亮度=0

5

🔄 下一回合 或 游戏结束

若未分出胜负 → 保存进度 → runTurn() · 若胜负已分 → 显示弹窗

三、核心数据结构（GameState）

const GameState = {
  // === 进度 ===
  round: 0,                // 当前回合（1~20）
  level: 1,                 // 当前关卡（1~5）
  phase: 'LIGHTHOUSE',      // 当前阶段

  // === 风暴压力 ===
  stormPressure: 1,          // 当前压力（0~5）

  // === 资源 ===
  resources: {
    light: 5,               // 亮度（0~10）
    structure: 10,           // 结构值（0~10）
    victoryPoints: 0,         // 胜利点数（0~∞）
    dreamSoothers: 0          // 抚慰物数量（0~5）
  },

  // === 灯塔 ===
  lighthouse: {
    position: 0,            // 轨道位置（0~40）
    maxStructure: 10
  },

  // === 风暴 ===
  storm: {
    eyePosition: 20,        // 风暴之眼位置（0~40）
    tiles: [],               // 风暴格子数组
    gameOver: null | 'WIN_VP' | ...
  },

  // === 船员（5名）===
  crew: [
    { id: 1, name: '船员A', status: 'IDLE', skill: null },
    // ... 其他4个船员
  ]
};
